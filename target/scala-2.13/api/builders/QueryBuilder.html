QueryBuilder

abstract class QueryBuilder extends AnyRef

Defines behaviour for building any type of a query string. Contains three lists -

  • scalars - list of strings that represent a scalar in GraphQL
  • fields - list of QueryBuilder that contains other scalars and connections
  • connections - list of QueryBuilder that contains nodes of a specific models.objects.GraphQLObject

Defines a method called construct which takes the included scalars/fields/connections and constructs the query string by iterating through them recursively. Extended by -
